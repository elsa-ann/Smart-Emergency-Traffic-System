# Web Prototype — SECS

This folder contains the interactive prototype and Streamlit dashboard.

## Files
- `app.html` — HTML5 canvas prototype (emergency corridor simulation)
- `app.py` — Streamlit launcher with view selector
- `app_sim.py` — Simulator mode for dataset playback and scenario testing

## How to run
1. Create venv and install requirements
2. `streamlit run Web_Prototype/app.py`

## Views
- Prototype (HTML): opens the canvas simulation
- Simulator: dataset-based scenario player
- Streamlit Dashboard: AI reasoning and scan interface
- Webcam AI: demo webcam capture
