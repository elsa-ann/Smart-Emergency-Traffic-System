# Smart-Emergency-Traffic-System

This repository contains the Smart Emergency Corridor System (SECS): an AI-assisted prototype to detect emergency vehicles and manage traffic signals to open emergency corridors.

Deliverables in this repo:

- Prototype Application: `Web_Prototype/app.html`, `Web_Prototype/app.py`, `Web_Prototype/app_sim.py`
- Poster (A1): `Documentation/POSTER_A1.md` (draft)
- Source Code: core scripts under `AI_Model/`, Streamlit apps under `Web_Prototype/`
- Dataset: `Dataset/` (traffic_decisions.csv, bandung-cctv, emergency-vehicles)
- User Guide & Ethical Assessment: `Documentation/USER_GUIDE.md`

See `Documentation/` for guardrails, bias analysis, and contribution guidelines.
